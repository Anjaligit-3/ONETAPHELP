One Tap Help (SOS System)

A real-life inspired emergency web application built using HTML, CSS, and JavaScript that allows users to send an instant SOS alert with a single tap.

Features
One-tap SOS alert button
Direct WhatsApp emergency message
Browser notification alert
Responsive and user-friendly interface
Anti-spam button control (10-second cooldown)
Tech Stack
HTML
CSS
JavaScript (Vanilla JS)

How It Works

The user clicks the SOS button
A browser notification is triggered
WhatsApp opens with a pre-filled emergency message
A status message confirms that the alert has been sent
